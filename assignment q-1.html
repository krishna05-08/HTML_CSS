Q-1 Define the terms: Website, Webpage, Web browser, Web server, HTML,CSS

Answer:
Website: A collection of related webpages that are typically identified with a common domain name and accessible via the internet.
Webpage: A document or resource that is displayed on the World Wide Web, typically written in HTML (Hypertext Markup Language), containing text, images, multimedia elements, and hyperlinks.
Web browser: A software application used to access information on the World Wide Web. Examples include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari.
Web server: A computer program or software that delivers content, such as webpages, to clients (typically web browsers) over the internet using the Hypertext Transfer Protocol (HTTP).
HTML (Hypertext Markup Language): The standard markup language for creating webpages and web applications. It defines the structure and layout of content on a webpage using tags and attributes.
CSS (Cascading Style Sheets): A style sheet language used for describing the presentation and formatting of a document written in HTML or XML. CSS defines how HTML elements should be displayed on the screen, in print, or in other media.
